How it Works

Up and running in 3 steps

1

Download the app

Click "Download Free App" to save a single DailyExpensesTracker.html file to your computer. No installation required — just open it in Chrome or Safari.

2

Import your bank statement

Export your DBS/POSB CSV from digibank, then go to Settings → Import Jan–Jun 2026 Data. 392 transactions auto-categorized in under a second.

3

Track, review, and share

Add daily expenses, view monthly summaries, export CSV reports, and share the app file with anyone — your family, partner, or team. No accounts, no subscriptions.

Deploy Online

Host it free — share a link

Two minutes to deploy. Get a shareable URL anyone can open from any device.

GitHub Pages
Free · Custom domain support
1
Go to github.com/new → Create a public repo named daily-expenses-tracker
2
Click Add file → Upload files → drag in index.html, webapp.html → Commit
3
Go to Settings → Pages → Source: Deploy from branch → Branch: main → Save
4
Your live URL: https://[username].github.io/daily-expenses-tracker/
⏱ Takes ~2 min to go live after enabling Pages
Supabase Storage
Free tier · 1GB storage
1
Go to supabase.com/dashboard → New project → give it a name
2
Sidebar → Storage → New bucket → name app → ✅ Public bucket
3
Upload index.html and webapp.html → click each file → Copy URL
4
Share the public URL: [project].supabase.co/storage/v1/object/public/app/webapp.html
💡 Best for sharing the app file directly. GitHub Pages gives a cleaner URL.
